## About

The First Documentation Theme

## Installation

```bash
$ npm i egg-born-module-cms-themedocs
```

## Usage

Changing languange's theme to `cms-themedocs`

```javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themedocs"
}
```
