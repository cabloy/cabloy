EggBorn.js Showcase: Watch Articles

这是EggBorn.js的案例，同时也是一个强大的工具，可以自动抓取文章的统计数字，诸如visits、 stars、 replies，等等。

Vue.js + Framework7 + Koa.js + Egg.js + MySQL + GitHub Passport

主要演示特性

  • 模块异步加载
  • Framework7 Tabs View布局
  • 全局Indicator显示与隐藏
  • “上拉刷新”与“下拉加载”
  • LoadMore组件设计
  • Form Validate
  • Login页面跳转逻辑
  • GitHub Passport整合
