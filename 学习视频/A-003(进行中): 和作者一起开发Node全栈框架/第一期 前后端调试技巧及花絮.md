## 如何定位代码逻辑

### 如何定位某个按钮的代码逻辑？

1. 命令行输入：window.\_\_debugger=true
2. Dom 节点：添加 debugger 属性

### 如何定位 API 的调用位置？

- 进入 F12 - Network 面板
- 查看 API 的 Initiator 字段值

### 如何查看页面组件的路由及代码位置？

- 直接查看 Page 对应 Dom 节点的属性值：
  - data-dev-page-url

### 如何查看 Vue 组件的代码位置？

- 直接查看组件对应 Dom 节点的属性值：
  - data-dev-component-module
  - data-dev-component-tag
